# Insulated Roof Panel RFQ Toolkit

Open-source procurement templates and checklists for insulated aluminum roof panels.

These tools help buyers prepare a project-ready RFQ. Product-specific values must be confirmed with the quoted factory documents and local engineering/import requirements.

## Request a quotation

For a project-specific quotation, send the country or destination port, approximate area, building use, preferred configuration, drawings and required documents through the [GSM BuildTech contact form](https://insulatedaluminumroof.com/contact/?utm_source=github&utm_medium=opensource&utm_campaign=toolkit_contact).

## Scope

This repository is for procurement preparation. It does not certify a product or guarantee price, stock, MOQ, delivery, fire rating, U-value or span.
