# Cold Storage Panel Buying Checklist

**Primary search intent:** `cold storage panel buying checklist`

A buyer checklist for cold-storage roof and wall-panel RFQs, including temperature objective, joints, condensation, drawings and required evidence.

## What this project contains

- A reusable procurement template or decision framework
- Clear fields for country/port, area, building use, configuration, drawings and required documents
- An evidence boundary: product-specific values must come from the quoted factory TDS, drawings or test records
- A buyer-friendly route to request a project-specific quotation

## Request a quotation

Use the [GSM BuildTech project RFQ form](https://insulatedaluminumroof.com/cold-storage-insulated-roof-panels/?utm_source=github&utm_medium=opensource&utm_campaign=cold-storage-panel-buying-checklist) and include the project area, application, destination port, preferred thickness/configuration, drawings and document requirements.

## Scope and limitations

This open-source checklist is for procurement preparation. It does not certify a product, establish a fire classification, guarantee a U-value, span, price, MOQ, stock level or delivery time, and does not replace local engineering or importer review.
